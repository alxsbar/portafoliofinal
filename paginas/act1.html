¿Qué son ? ¿Para qué sirven? 5 ejemplos de Emmet . Editores: Visual Estudio, Brackets, Atom, Sublime Text. Complemento Emmet.

Publicado 9 de octubre de 2023

Emmet es una herramienta para desarrolladores web que ayuda a acelerar la escritura de HTML y CSS mediante la utilización de abreviaturas y atajos. Al escribir estas abreviaturas y presionar una tecla específica (a menudo Tab), Emmet las expandirá automáticamente al código correspondiente. ¿Para qué sirve Emmet? Rapidez: Reduce significativamente el tiempo necesario para escribir estructuras de código repetitivas. Menos errores: Al escribir menos, hay menos oportunidades de cometer errores tipográficos. Lectura simplificada: Las abreviaturas de Emmet son fáciles de leer y entender una vez que te familiarizas con ellas. Personalización: Puedes definir tus propios atajos y preferencias en Emmet. Editores y complemento Emmet: Visual Studio Code: Emmet está integrado por defecto. Brackets: Emmet necesita ser instalado como una extensión. Atom: Emmet se puede instalar a través del gestor de paquetes de Atom. Sublime Text: Emmet está disponible como un paquete que se puede instalar usando el gestor de paquetes de Sublime.

HTML5. Qué es y línea del tiempo

HTML5 es la quinta versión del lenguaje de marcado de hipertexto (HTML), que es el estándar utilizado para estructurar y presentar contenido en la web. Esta versión introdujo muchas mejoras y características nuevas en comparación con sus predecesoras, haciéndolo más potente, flexible y adecuado para la creación de aplicaciones web modernas y sitios web responsivos. **Línea del Tiempo Resumida de HTML5**: 1. **2004**: WHATWG inicia el desarrollo de una actualización de HTML. 2. **2008**: W3C adopta el trabajo de WHATWG y comienza a trabajar en HTML5. 3. **2012**: Primer borrador completo de HTML5 publicado. 4. **2014**: HTML5 se establece oficialmente como norma de W3C. 5. **2016 y más allá**: W3C adopta una estrategia de desarrollo continuo, incorporando cambios y actualizaciones de manera regular en lugar de esperar a nuevas versiones.

CSS3. Qué es ? Sintaxis

**CSS3**: CSS3 es la tercera versión del lenguaje de hojas de estilo en cascada (CSS). Se usa para definir la apariencia y diseño de documentos escritos en HTML y XML. A diferencia de sus predecesores, CSS3 se divide en "módulos", proporcionando especificaciones más detalladas y claras para cada aspecto del diseño y comportamiento en la web. **Sintaxis Básica**: La estructura de CSS3 se basa en definir reglas de estilo. Cada regla tiene un selector y un conjunto de propiedades definidas entre llaves. Cada propiedad se separa de su valor por dos puntos, y las propiedades individuales se separan por punto y coma. ```css selector { propiedad: valor; } ``` Por ejemplo, si deseamos cambiar el color de todos los párrafos a rojo: ```css p { color: red; } ``` Esta estructura es la base de cómo se aplica el diseño y la apariencia en la web mediante CSS.

**Bootstrap**: Bootstrap es un marco de diseño (framework) de código abierto orientado al desarrollo de sitios web y aplicaciones móviles. En que consisten Diseño: Responsive Web y UX

**Diseño Responsive Web (RWD):** Es un enfoque que permite que un sitio web se adapte y se visualice correctamente en diferentes dispositivos, desde computadoras hasta móviles. Utiliza rejillas flexibles, imágenes adaptables y media queries para ajustar el contenido según el tamaño de pantalla. **Diseño de Experiencia de Usuario (UX):** Se centra en la interacción y satisfacción del usuario con un producto o servicio. Implica investigar las necesidades del usuario, estructurar la información, diseñar interacciones, crear prototipos y realizar pruebas de usabilidad para asegurar una experiencia positiva y coherente. Ambos enfoques buscan garantizar que los sitios web y aplicaciones sean accesibles, intuitivos y atractivos para los usuarios en diferentes dispositivos y contextos.
