Desarrolla front-end like a boss aprendiendo Bootstrap 3

En este workshop aprenderás...

En este workshop de 7 horas de duración Marcos González y Armando Sotoca te mostrarán como optimizar el flujo de trabajo a la hora realizar el front-end de un proyecto web. La metodología apoyada en el framework Bootstrap 3 parte desde la fase de diseño hasta finalizar en la fase de desarrollo. Aprenderás a diseñar Bootstrap, a orientar el diseño bajo la filosofía mobile first y responsive web design y descubrirás la metodología adecuada para convertir ese diseño en maqueta HTML5 de una forma ágil y sencilla.

Como siempre podrás complementar todo lo que aprendas durante el workshop compartiendo experiencias con otros asistentes y el /los ponentes durante la comida o la cena típica austriaca que degustarán todos los que se animen.
